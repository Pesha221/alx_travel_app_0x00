## Overview
This project provides CRUD API endpoints for managing Listings and Bookings using Django REST Framework. The API is fully documented using Swagger.

## Endpoints
All API endpoints are available under `/api/`.

### Listings
- GET `/api/listings/`
- POST `/api/listings/`
- GET `/api/listings/<id>/`
- PUT `/api/listings/<id>/`
- DELETE `/api/listings/<id>/`

### Bookings
- GET `/api/bookings/`
- POST `/api/bookings/`
- GET `/api/bookings/<id>/`
- PUT `/api/bookings/<id>/`
- DELETE `/api/bookings/<id>/`

## Swagger Documentation
Available at:

- `/swagger/`
- `/redoc/`

## Tools Used
- Django
- Django REST Framework
- drf-yasg (Swagger documentation)
- Postman
